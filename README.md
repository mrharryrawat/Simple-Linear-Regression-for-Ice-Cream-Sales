# Simple-Linear-Regression-for-Ice-Cream-Sales
Ice Cream Revenue Prediction
#Overview

This project uses Machine Learning (Linear Regression) to predict ice cream shop revenue based on the temperature. The application is deployed using Flask, allowing users to enter a temperature value and receive a predicted revenue instantly through a web interface.

#Features
Predicts ice cream revenue from temperature input.
Simple and user-friendly web interface.
Built using Flask and Linear Regression.
Displays predictions on the same webpage.
Dataset Information

#The dataset contains historical records of temperature and the corresponding ice cream revenue.

Input Feature: Temperature
Target Variable: Revenue

#The model learns the relationship between temperature and revenue and uses it to predict revenue for new temperature values.
